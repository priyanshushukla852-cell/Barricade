# Barricade – Mobile Game (React Native / Expo)

A 2-player digital Barricade board game. Red vs Blue. Each player moves one piece across a 9×9 grid toward the opponent's starting row, spending turns placing wall segments to block movement. Supports local (hot seat), VS computer (easy/hard AI), and online real-time multiplayer with a configurable per-turn chess clock.
